
Warum?
- Inhalte im Spiel anzeigen, die vom Server kommen, ohne dass man
  dafür den Browser öffnen muss

Anforderungen:
- wenige: CSS wäre cool aber es reicht, wenn man den Style per Code
  beeinflussen kann

  sich danach richten, was der Browser kann
